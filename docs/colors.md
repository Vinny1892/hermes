|-------|-----|-----|

| `--text` | `#e4e4f6` | Texto corrido |
| `--text-muted` | `#acaccc` | Labels, legendas, texto secundário |
| `--text-bright` | `#fafafd` | Títulos, headings |
| `--error` | `#ff4455` | Erros, mensagens de falha |
